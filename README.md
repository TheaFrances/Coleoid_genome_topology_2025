This repo contains code for the analyses presented in the paper "Genome reorganisation and expansion shape 3D genome architecture and define a distinct regulatory landscape in coleoid cephalopods" — Thea F. Rogers, Jessica Stock, Natalie Grace Schulz, Gözde Yalçin, Simone Rencken, Anton Weissenbacher, Tereza Clarence, Darrin T. Schultz¹, Cliff Ragsdale, Carrie Albertin, Oleg Simakov (2025).

**Note**: The category "Not interacting in any species" used in the R scripts corresponds to "No conserved coleoid interaction" as referred to in the manuscript and throughout the github READMEs.

Species abbreviations used across scripts and plots:
- *Euprymna scolopes*: eup or eupsc
- *Sepia officinalis*: sof or sepof
- *Octopus bimaculoides*: obi or octbi
